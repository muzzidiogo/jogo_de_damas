                              JOGO DE DAMAS
                              
  * Para se usar o menu aperte duas vezes a letra inicial de cada 
    comando (uma para ativar e outra para confirmar):
      s:Sair    i:Info    r:Reiniciar/Retornar

  * Para fazer a jogada digite 'j' e escreva o comando:

      'Xpeça Ypeça Xfinal Yfinal' : para mover a peça deve se escrever 
                                    as cordenadas no tabuleiro para 
                                    o local inicial e final da peça 
                                    com numeros inteiros

      '!' : sair do Modo Jogada

                  Para mais informações, leia o README.md
